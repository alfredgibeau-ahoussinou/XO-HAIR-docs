# Contribuer à la documentation

Ce dépôt est **public** : n'y ajoutez jamais :

- clés API, mots de passe, `ADMIN_CODE`
- fichiers `.env` ou captures d'écran de Netlify avec secrets
- données clients (emails, téléphones, exports CSV)

## Modifications acceptées

- Mise à jour des horaires / processus métier
- Clarifications pour Ola en français
- Liens vers le site ou Instagram

Proposez vos changements via **pull request** ou demandez l'accès d'écriture au mainteneur.
